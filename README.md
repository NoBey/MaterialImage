# MaterialImage
Working!

## 使用方法

### ES6 Module:
``` javascript
import MaterialImage from 'material-image'; // 尚未发布至npm

const body = document.querySelector('body');
MaterialImage(body);
```

### Script:
``` html
<script src="materialImage.min.js"></script>
<script>
    var body = document.querySelector('body');
    MaterialImage(body);
</script>
```

## 生成策略

随机一定数量的颜色，再随机画出不同尺寸、位置的图形（矩形和圆）。
